Installation
==========

## 1. Install CocoaPods
```console
$ sudo gem install cocoapods
```

## 2. Install Carthage with [Homebrew](https://brew.sh/)
```console
$ brew install carthage
```

## 3. Run script
```console
$ cd scripts/dependency
$ sh lib_install.sh
```

## 4. Open Xcode
```console
$ cd ../..
```
```console
$ open Cashdoc.xcworkspace/
```
or
```console
$ xed .
```
